安装

系统要求：Debian/Ubuntu系统，内核版本v4.9.3-v4.12.x。得安装gcc，Makefile默认只支持gcc 6，你可以修改它（例如）gcc-4.9。
安装
系统要求：Debian/Ubuntu系统，内核版本v4.9.3-v4.12.x。得安装gcc，Makefile默认只支持gcc 6，你可以修改它（例如）gcc-4.9。
