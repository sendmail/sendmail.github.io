CPUS:将老打印机变为网络打印机,且支持 苹果机的AirPrint

IPP打印很好用,
支持IOS AirPrint, 手机直接打印文档,最主要是老婆给小孩子 打印作业,省得老是开电脑了.

apt-get install hannah-foo2zjs printer-driver-foo2zjs-common printer-driver-foo2zjs

getweb 1018 (取hp1018驱动)

apt-get install cups
修改 cups的配置 nano /etc/cups/cupsd.conf
重启cups service cups restart


win 添加打印机:
http://192.168.0.88:631/printers/HP_1018

cups支持的打印机
http://www.openprinting.org/printers

airprint: 中文说明
http://blog.itist.tw/2015/05/porting-airprint-service-with-rapberry-pi.html

sudo apt-get -y install avahi-daemon avahi-discover libnss-mdns

sudo service avahi-daemon restart

一定要手动重启avahi才能刷新bonjour :
https://bugs.launchpad.net/ubuntu/+source/avahi/+bug/624043

解决修改 /etc/rc.local
sleep 30
service avahi-daemon restart

延时启动就解决了.
