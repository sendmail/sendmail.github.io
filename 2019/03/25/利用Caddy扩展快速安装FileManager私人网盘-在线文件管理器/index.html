Caddy FileManager扩展介绍
FileManager是基于Caddy的扩展。它提供文件管理界面，可用于上传/下载/删除/预览和重命名等该目录中的文件。

支持 上传文件

支持 按类型 搜索文件

支持 批量压缩 文件下载

支持 多用户管理(权限可控)

支持 在网页执行 Linux命令

支持 创建 共享链接(限时/永久)

支持 在线编辑 各类文本文件

支持 在线浏览 图片/文本/视频等

支持 新建/重命名/移动/删除 文件和文件夹等
