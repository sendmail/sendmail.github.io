CDN说明
嵌套cloudflare(fullSSL+CDN)节省资源、人人有责、如果你的ip没有被Q，请勿开启。

提前准备购买域名、VPS；
注册cloudflare，提前24小时配置NS并解析域名。
确保所解析的域名Status位置关闭默认的cdn加速，变为灰色云朵状态。
开启cloudflare(fullssl+cdn)安装完毕一键脚本，并测试V2可以正常使用之后。
注意：在此步骤之前，勿必保持所解析的域名Status处为灰色云朵（及关闭cdn）。重做系统或重新安装时也勿必关闭此处设置。

  1. 开启全程ssl加密
    在Cloudflare的Crypto那里设置SSL为Full。
  2. 开启cdn隐藏ip地址
    返回Status点击灰色云朵图标 将其变成橙色云朵状态开启cdn，即是DNS and HTTP proxy(CDN)
  3. 开启Always use HTTPS
    开启http强制https跳转
